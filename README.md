# B2B Sales Intelligence Dashboard

A machine learning based B2B Sales Intelligence Dashboard built using Python and Streamlit. It analyzes sales data and predicts future sales using a regression model.

## Features
- Sales data preprocessing
- Interactive charts and graphs
- Sales trend analysis
- Future sales prediction
- Streamlit dashboard UI

## Tech Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib / Plotly

## How to Run
Install libraries:
pip install streamlit pandas scikit-learn matplotlib plotly

Run app:
streamlit run forecast_dashboard.py

## Files
- forecast_dashboard.py — main dashboard app
- Screenshots.pdf — output dashboard images
